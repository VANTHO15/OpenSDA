# Cách chạy
- Nhấn giữ nút SW5 và cắm USB vào laptop
- Khi nay sẽ hiện BOOTLOADER như 1 ổ cứng trên lap
- Kéo thả file OpenSDA_V1.bin vào Ổ cứng đó
- Nhổ USB và cắm lại, Tận hưởng thôi
